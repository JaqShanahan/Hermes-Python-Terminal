## About The Project

This is a basic python console application that provides a few functionalities. It uses the colorama library to add color to the output and the socket library to retrieve the host name and IP address of the system. The application accepts user inputs and performs different actions based on the input provided. Currently the actions include:

- Displaying the IP address of the system
- Displaying the message passed in between double quotes after the "display" keyword
- Typing out the message passed in between double quotes after the "typeOut" keyword
- Clearing the console screen
- Performing basic mathematical operations such as addition, - subtraction, multiplication, and division
